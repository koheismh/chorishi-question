# questions.json レビュー結果

調理師試験対策問題（全240問、各問 basic / exam の2形式＋解説）を対象に、
「誤った知識・誤った正答が含まれていないか」を最重要観点としてレビューした。

## 更新履歴（「試験に合格する」観点で修正済み）

知識として誤り・古い・内部不整合のものだけを修正した。正答に影響しない選択肢の
作り込み・重複、および調理師試験の慣用表記が優先される項目は変更していない。

- **id 233**：食品ロス量「約500万トン以上」→「約472万トン（令和4年度）」に更新（古いデータの是正）。
- **id 231**：解説の「知恵がつくのは栗きんとん（勝ち栗）」を削除し、「海老＝長寿、知恵の意味はない」に修正。
- **id 179**：解説のカラメル化温度を「165℃以上」→「170℃以上」に統一（exam選択肢の170℃と整合）。
- **id 72**：選択肢「AとB とC」の余分な空白を「AとBとC」に修正。

### 合格観点で「変更しない」と判断した項目
- **id 146（茶の「発酵」）**：厳密には酸化だが、調理師試験では「発酵度による分類」が慣用。試験の傾向を正とし据え置き。
- **id 162（卵黄凝固温度 65〜70℃）**：試験で用いられる温度帯表記の範囲内のため据え置き。
- **カテゴリD（識別力の低い選択肢）・カテゴリE（重複）**：誤答ではなく、合格には影響しないため据え置き。

## 総評

- 解答（answer）はすべて **0始まりのインデックス** で統一されており、抽出した範囲で **正答キーの明確な誤り（正解そのものが間違っている）は発見されなかった**。
- 科学的・法制度的な記述も概ね正確で、全体として完成度は高い。
- ただし、以下のとおり「知識・数値の精度」「試験問題としての適切性」「重複」「表記」の観点で改善余地がある。

指摘は重要度の高い順にカテゴリ分けして記載する。

## カテゴリA：知識・データの誤り／古さ（要確認・修正）

### A-1. id 233（食文化概論）食品ロスの数値が古い
- exam選択肢「日本の食品ロスは年間約500万トン以上である」を**正しい前提**で出題している。
- 近年の農林水産省・環境省推計では食品ロス量は減少傾向で、直近年度は **約470万トン台（令和4年度＝約472万トン）** と、500万トンを下回っている。
- 「約500万トン前後（年度により変動）」等、現状に合わせた表現への修正を推奨。

### A-2. id 231（食文化概論）解説の意味付けが不正確
- 「海老＝長寿」という核心の正誤判定は正しい。
- ただし解説中の **「知恵がつくのは栗きんとん（勝ち栗）」は不正確**。栗きんとんは一般に「金運・財運」、勝ち栗は「勝負運」の縁起物であり、「知恵」を表す定番のおせちは存在しない。
- 解説からこの一文を削除、または「海老は長寿の縁起物である」点のみを述べる形に修正を推奨。

## カテゴリB：内部の数値不一致・精度（軽微）

### B-1. id 179（調理理論）カラメル化温度の閾値不一致
- exam選択肢：「砂糖を**170℃以上**に加熱するとカラメル化する」
- 解説：「**165℃以上**でカラメル化が起こる」
- 同一問題内で閾値が食い違っている。どちらかに統一（160〜170℃の範囲表記でも可）を推奨。

### B-2. id 162（調理理論）卵黄の凝固開始温度
- basic「卵黄が凝固し始める温度」を「65〜70℃」としている。
- 一般に卵黄は **約65℃で凝固し始め、約70℃でほぼ完全凝固**。範囲表記のため誤りとは言い切れないが、「凝固し始める」と「完全凝固」が混在しやすい。表現の明確化を検討。

## カテゴリC：用語の科学的厳密性（許容範囲・参考）

### C-1. id 146（食品学）茶の「発酵」
- 「緑茶・ウーロン茶・紅茶は発酵度の違いで分類される」は、厳密には酵素による **酸化** 反応。
- ただし調理師教本では慣用的に「発酵」と表記されるため、試験対策上は許容範囲。参考情報として記載。

## カテゴリD：試験問題としての適切性（識別力の低い設問）

誤答選択肢が明らかに不自然・価値判断的で、知識がなくても常識で正答できてしまう設問群。
特に **食文化概論** に集中している。試験（exam）としての識別力が弱いため、選択肢の作り込みを推奨。

- id 209（盛り付け）：「食べにくくする」「料理を冷ます」等、誤答が非現実的。
- id 233（食品ロス）：basic選択肢「食料が余っているから」等が不自然。
- id 235（器）：「和食では使い捨て容器が伝統的に推奨される」等、明白に消去可能。
- id 236（地産地消）／id 238（もったいない）／id 240（調理師の役割）：
  正答が価値判断的に自明で、誤答（利益追求のみ／大量廃棄 等）が機能していない。

また、以下は「すべて〜」「全く〜ない」型の極端な断定で容易に消去できる誤答パターン。
試験では一定数許容されるが、過多になると難易度・識別力が下がる。
- id 229「日本は食料をほとんど輸入に頼っていない」
- id 237「すべての国の食文化は同じ調味料を使用している」
- id 239「日本人の食生活は戦後から現在まで全く変化していない」
